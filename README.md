# 🛠️ kleeDebugger

**符号执行可视化调试工具**

### 简介

本项目是一个基于 KLEE 的[符号执行](https://en.wikipedia.org/wiki/Symbolic_execution)可视化调试插件，集成于 [VS Code](https://code.visualstudio.com/) 开发环境，旨在为 C/C++ 程序提供更加直观、交互式的符号执行体验。
